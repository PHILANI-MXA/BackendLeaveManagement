My endpoints for my API


To get all employees

method = get: http://localhost:3000/employees

To get a single employee

method = get: http://localhost:3000/:id

To get leave requests

method = get: http://localhost:3000/LeaveRequests

To register employees

method = post: http://localhost:3000/post/employee

To delete employees

method = delete: http://localhost:3000/id

To delete leave applications

method=delete: http://localhost:3000/id

To apply for a leave

method = post: http://localhost:3000/:id/application
